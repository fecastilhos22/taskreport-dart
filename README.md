TaskReport Dart
📋 Sobre o Projeto

O TaskReport Dart é uma aplicação desenvolvida em Dart que simula o processamento de dados retornados por uma API de gerenciamento de tarefas.

O sistema recebe uma lista de mapas contendo informações das tarefas, realiza o tratamento e a conversão dos dados, transforma os registros em objetos e gera relatórios gerenciais com base nos status, prioridades, horas trabalhadas e valores das tarefas.

O projeto foi desenvolvido como atividade prática do Módulo 01 do curso Desenvolvedor(a) Mobile, com o objetivo de aplicar conceitos fundamentais da linguagem Dart e boas práticas de organização de código.

🎯 Objetivos
Simular o consumo de dados provenientes de uma API;
Realizar conversão e tratamento de dados;
Aplicar Programação Orientada a Objetos;
Gerar relatórios utilizando coleções e métodos da linguagem;
Praticar Git, GitHub e GitFlow;
Organizar atividades utilizando Kanban.

🚀 Funcionalidades Implementadas
Conversão de dados de Map para objetos;
Tratamento de valores nulos;
Conversão de Strings para números;
Limpeza e padronização de dados;
Listagem de tarefas pendentes;
Cálculo de valores totais;
Cálculo de médias;
Agrupamento de tarefas por status;
Contagem de tarefas por categoria;
Identificação de tarefas com dados incompletos;
Geração de relatório final no console.
